# MotoHire

Car rental booking application

# About the Project

This is car rental booking website build from scratch using MERN.
This web application is well integrated with many modern features.

## Functionalities

User Side

- Responsive Design
- OTP Login
- Wishlist
- Map Navigation
- Chat Bot
- Search
- Coupon
- Offer
- Booking History
- Payment using Paypal and Razorpay
- Filter Cars

Admin Side

- User Management
- Car Management
- Coupon, Offer 
- District Management
- Booking Management
